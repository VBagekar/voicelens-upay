# 🎙️ VoiceLens – UPAY NGO Sentiment & Impact Analyzer

> Empowering UPAY NGO with AI-driven insights from community voices.

---

## 🌟 What is VoiceLens?

VoiceLens is an AI-powered sentiment and impact analysis tool built specifically
for UPAY NGO. It analyzes feedback from volunteers, beneficiaries, and community
members — turning raw text into meaningful, actionable insights.

Whether it's a survey response, a social media comment, or a program review,
VoiceLens helps UPAY understand the **emotional pulse** of their community.

---

## 🎯 Key Features

- 📝 **Text Sentiment Analysis** — Analyze single feedback entries
- 📂 **Bulk File Analysis** — Upload CSV/Excel files of responses
- 💬 **Comment Analyzer** — Analyze social media or survey comments
- 📊 **Impact Dashboard** — Visual summary of emotional trends
- 🤗 **Multi-model Support** — VADER, TextBlob, and BERT (local only)
